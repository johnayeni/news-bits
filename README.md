# news-bits

This is a demo app for a tutorial.

### Installing

Dependencies:

* [Node.js](https://nodejs.org/en/) v8.9.4 or above

Then check out the project and run:

```sh
npm install
```

### Running

```sh
npm run serve
```

### Using the app

You should now have the app server at [localhost:3000](http://localhost:3000)
